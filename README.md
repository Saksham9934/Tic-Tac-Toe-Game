# Tic-Tac-Toe-Game

**Tic Tac Toe Game**

A simple Tic Tac Toe game built using HTML5, CSS, and JavaScript.

**Features:**

- Play as X or O
- Alternate turns between players
- Win, lose, or draw detection
- Responsive design

**Technologies:**

- HTML5
- CSS
- JavaScript


**Usage:**

Open index.html in your web browser to play the game.
